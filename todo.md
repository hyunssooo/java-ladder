## 계산기 테스트 목록
----
* 덧셈 테스트
    * "1,1" -> 2
    * "1,1,1" -> 3
* 기본 구분자 테스트
    * "1:1" -> 2
* 커스텀 구분자 테스트
    * "//;\n1;1" -> 2
* 복합구분자(구분자가 여러개 들어왔을 때) 테스트
    * "1,1:1" -> 3
* 텍스트 필터링 테스트
    * //adf\n1adf2adf3adf4 -> 1adf2adf3adf4
* 음수가 넣었을 때 예외를 던지는지 테스트  
    * "-1,1" -> 예외 던지기
* 숫자 이외의 값을 넣었을 때 예외를 던지는지 테스트  
    * "-1,a" -> 예외 던지기
* 커스텀 구분자가 숫자인 경우 예외 테스트
    * "1" -> 예외 던지기
